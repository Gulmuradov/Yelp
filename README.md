## Installation

Install ReactJs App, Amplify Project

```
  npx create-react-app .
  npm i -g @aws-amplify/cli@4.24
```

## Run Locally

Go to the project directory

```bash
  cd yelp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
